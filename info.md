# ClickUp Todo Card

An enhanced custom Lovelace card for Home Assistant that displays ClickUp tasks with full support for custom fields, priorities, tags, assignees, and advanced filtering.

## Features

- Display ClickUp custom fields (14 types supported)
- Show both start dates and due dates
- Multiple/custom status support from ClickUp
- Priority indicators with colors
- Tags with custom colors
- Assignee avatars
- Client-side filtering by status, priority, tags, assignees, and date ranges
- Sort by due date, start date, priority, name, or status
- Group tasks by status, priority, assignee, or custom field
- Compact mode for smaller displays
- Visual configuration editor
- Works with both list and view entities from the ClickUp integration

## Requirements

- Home Assistant with Lovelace UI
- [ClickUp Integration](https://github.com/chbarnhouse/ha-clickup) installed and configured

## Quick Start

1. Install this card via HACS
2. Add the card to your Lovelace dashboard
3. Select a ClickUp todo entity
4. Configure display options and filters as desired

For detailed installation and configuration instructions, see the [README](https://github.com/chbarnhouse/clickup-todo-card).
